# LLM-Pseudoword-Acquisition
# LLM-Pseudoword-Acquisition > 基于 Llama-Factory 的大模型伪词（Pseudoword）习得机制与语义表征实验  ## 🌟 项目简介 (Project Overview) 本项目是一个结合**计算语言学 (Computational Linguistics)** 与 **大模型微调 (LLM Fine-tuning)** 的交叉实验。  核心目标是探究小参数语言模型（以 Qwen-8B 为例）在接触全新音位合法（Phonotactically legal）但无先验语义的“伪词”时，如何通过不同维度的指令数据构建语义映射，并观测其从“接受性知识”到“产出性知识”的转化效率。
